# Refresh-scale balanced Adam experiments

This repository contains the code and compact derived data needed to reproduce
the figures and tables used in the paper.

The experimental setup studies balanced Adam, where

```text
beta_1 = beta_2 = beta
```

and interprets `H_beta = 1 / (1 - beta)` as the optimizer memory horizon.  The
paper uses the empirical refresh rule

```text
R_beta = (1 - beta) T_ES ~= 1000
```

to select one beta value per experiment.

## Repository layout

```text
github_ready/
  training/
    experiments/       Training scripts and dataset helpers.
    README.md          Exact training protocols and seed policy.
    beta_grid.txt      Common beta grid.
    experiment_index.tsv
  analysis/
    plot_beta_sweep_curves.py
    make_paper_tables.py
    README.md
  data/
    beta_sweep_curves.tsv
    table_selected_betas.tsv
    table_main_results.tsv
    table_beta_stability_intervals.tsv
    table_refresh_noise_robustness.tsv
  outputs/
    figures/           Paper figures, regenerated by analysis scripts.
    tables/            LaTeX tables, regenerated by analysis scripts.
  requirements.txt
```

The repository intentionally does not include raw training logs or model
checkpoints.  The offline analysis uses compact derived TSV files, so the paper
figures and tables can be regenerated without downloading datasets or rerunning
training.

## Reproduce paper figures and tables

From the repository root:

```bash
python analysis/plot_beta_sweep_curves.py
python analysis/make_paper_tables.py
```

This writes:

```text
outputs/figures/beta_sweep_curves_train.pdf
outputs/figures/beta_sweep_curves_validation.pdf
outputs/tables/table_selected_betas.tex
outputs/tables/table_main_results.tex
outputs/tables/table_beta_stability_intervals.tex
outputs/tables/table_refresh_noise_robustness.tex
```

## Experiments shown in the paper

The development set contains eight experiments: ResNet50 on Food-101 and
ImageNet100, ViT-B/16 on CIFAR-100 and TinyImageNet, NanoGPT on WikiText-103
and OpenWebText, and Llama60M on C4 and SlimPajama-6B.  The held-out set
contains three additional experiments: T5-small on BookCorpus, Swin-T on
Caltech-256, and EfficientNet-B0 on Stanford Cars.

## Notes

- All reported gaps are relative validation-loss gaps to the per-experiment
  validation oracle beta.
- The `train` or `development` block in the tables means the eight experiments
  used to calibrate `R_0`, not train-loss metrics.
- T5-small uses seed 1 only in the final analysis; extra T5 seeds were excluded
  from the derived curves.
